# 이메일 별칭 (alias) 기능 사용하기
- 구글의 alias 기능은 plus, dot을 사용자 이름에 추가해서 이메일을 보내면 원래 구글 메일 계정으로 받을 수 있다.

## 1. 사용자 이름에 점 (dot) 사용하기
- 점은 아래와 같은 형식으로 추가할 수 있다.
```
foo@gmail.com
f.oo@gmail.com
fo.o@gmail.com
f.o.o@gmail.com
```

## 2. 사용자 이름에 플러스 (+) 사용하기
```
foo@gmail.com
foo+aws1@gmail.com
foo+aws2@gmail.com
foo+us@gmail.com
```
- 위 모든 메일 주소는 모두 foo@gmail.com의 메일로 가게 된다.
